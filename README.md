# Peptide analysis

> **Project ID**: p02
>
> **Project Name**: peptide_analysis
>

This project performs statistical analysis on peptide data, including normality tests and Kruskal-Wallis tests, and generates visualizations of the results.

## Steps:

- s01: Dataframe cleaning
- s02: Zero thresholding
- s03: Imputation
- s04: Normality tests
- s05: Kruskal-Wallis test
- s06: Bonferroni and Benjamini-Hochberg corrections
- s07: Visualization
- s08: Analysis of RBD subgroups of signigicant peptides
- s09: PCA analysis
